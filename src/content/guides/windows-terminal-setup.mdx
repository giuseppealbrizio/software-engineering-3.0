Guida per configurare PowerShell 7 con autocompletamento, Oh My Posh e icone

---

<StepHeader number={1} title="Installa PowerShell 7" />

```powershell
winget install Microsoft.PowerShell
```

Chiudi e riapri Windows Terminal. Ora vedrai "PowerShell" come nuovo profilo.

---

<StepHeader number={2} title="Abilita esecuzione script" />

Apri PowerShell 7 **come amministratore** ed esegui:

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

Conferma con `Y` quando richiesto.

---

<StepHeader number={3} title="Installa Oh My Posh" />

```powershell
winget install JanDeDobbeleer.OhMyPosh
```

Testa che funzioni:

```powershell
oh-my-posh init pwsh | Invoke-Expression
```

---

<StepHeader number={4} title="Installa un Nerd Font" />

Necessario per visualizzare le icone nel prompt:

```powershell
oh-my-posh font install
```

Scegli **CaskaydiaCove** o **FiraCode**.

<InfoBox type="info" title="Configurazione Font">
Vai in Windows Terminal â†’ Settings â†’ Profiles â†’ Defaults â†’ Appearance â†’ Font face e seleziona il font installato (es. "CaskaydiaCove Nerd Font").
</InfoBox>

---

<StepHeader number={5} title="Installa Terminal-Icons" />

```powershell
Install-Module -Name Terminal-Icons -Repository PSGallery -Force
```

---

<StepHeader number={6} title="Crea il file $PROFILE" />

Questo file viene eseguito ad ogni avvio di PowerShell:

```powershell
New-Item -Path $PROFILE -Type File -Force
notepad $PROFILE
```

### Contenuto del $PROFILE

```powershell
# Oh My Posh
oh-my-posh init pwsh | Invoke-Expression

# PSReadLine - Autocompletamento
Set-PSReadLineOption -PredictionSource HistoryAndPlugin
Set-PSReadLineOption -PredictionViewStyle ListView
Set-PSReadLineKeyHandler -Key Tab -Function MenuComplete
Set-PSReadLineKeyHandler -Key UpArrow -Function HistorySearchBackward
Set-PSReadLineKeyHandler -Key DownArrow -Function HistorySearchForward

# Terminal Icons
Import-Module Terminal-Icons
```

Copia questo contenuto nel file e salva.

---

<StepHeader number={7} title="Imposta PowerShell 7 come default" />

**In Windows Terminal:**
Settings â†’ Startup â†’ Default profile â†’ seleziona "PowerShell"

**In VS Code:**
Apri settings.json e aggiungi:

```json
{
    "terminal.integrated.defaultProfile.windows": "PowerShell"
}
```

---

<StepHeader number={8} title="Riavvia e verifica" />

Chiudi tutto e riapri Windows Terminal. Verifica con:

```powershell
# Versione PowerShell (deve essere 7.x)
$PSVersionTable.PSVersion

# Versione PSReadLine (deve essere 2.2+)
Get-Module PSReadLine
```

<InfoBox type="warning" title="Troubleshooting">
Se PSReadLine non supporta `-PredictionSource`, aggiornalo manualmente:

```powershell
Install-Module PSReadLine -Force -SkipPublisherCheck
```
</InfoBox>

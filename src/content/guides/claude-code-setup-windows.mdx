<StepHeader number={1} title="Installa nvm-windows" />

nvm-windows Ã¨ un version manager per Node.js che permette di installare e gestire multiple versioni di Node senza problemi di permessi.

### Download

Scarica `nvm-setup.exe` dall'ultima release:

[github.com/coreybutler/nvm-windows/releases](https://github.com/coreybutler/nvm-windows/releases)

### Installazione

- Esegui `nvm-setup.exe`
- Accetta le opzioni di default
- Chiudi e riapri il terminale dopo l'installazione

<InfoBox type="success" title="Vantaggi di nvm-windows">
| Vantaggio | Descrizione |
|-----------|-------------|
| **Multiple versioni** | Switcha tra Node 18, 20, 22 a seconda del progetto |
| **Niente admin** | Installa pacchetti globali senza permessi elevati |
| **Aggiornamenti puliti** | Un comando per aggiornare Node |
| **Rollback facile** | Torna alla versione precedente se qualcosa si rompe |
</InfoBox>

---

<StepHeader number={2} title="Installa Node.js" />

Apri PowerShell e installa la versione LTS (Long Term Support) di Node.js:

```powershell
nvm install lts
nvm use lts
```

### Verifica l'installazione

```powershell
node --version
npm --version
```

<InfoBox type="success" title="Output atteso">
Dovresti vedere i numeri di versione, es. `v22.x.x` e `10.x.x`
</InfoBox>

---

<StepHeader number={3} title="Installa Claude Code" />

Con npm disponibile, installa Claude Code globalmente:

```powershell
npm install -g @anthropic-ai/claude-code
```

### Verifica l'installazione

```powershell
claude --help
```

<InfoBox type="success" title="Installazione completata!">
Il comando `claude` Ã¨ ora disponibile in qualsiasi terminale.
</InfoBox>

## Troubleshooting

<InfoBox type="warning" title="nvm non riconosciuto">
Chiudi completamente il terminale e riaprilo. Se non funziona, prova a fare logout/login da Windows.
</InfoBox>

<InfoBox type="warning" title="claude non riconosciuto dopo l'installazione">
Verifica che npm abbia installato correttamente:

```powershell
npm list -g @anthropic-ai/claude-code
```

Se il pacchetto Ã¨ presente, riavvia il terminale.
</InfoBox>

### Comandi utili per debug

```powershell
# Verifica quale Node stai usando
nvm list

# Verifica il PATH di npm
npm config get prefix

# Reinstalla Claude Code
npm uninstall -g @anthropic-ai/claude-code
npm install -g @anthropic-ai/claude-code
```

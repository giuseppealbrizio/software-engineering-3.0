# Roadmap

Feature e miglioramenti pianificati per il sito.

## Completate

- [x] **Dark/Light Mode** - Toggle tema con persistenza localStorage
- [x] **Tempo di lettura** - Stima automatica per ogni articolo
- [x] **Indice dei contenuti** - TOC collassabile per navigazione rapida
- [x] **Copy Code** - Pulsante copia sui blocchi di codice
- [x] **Back to Top** - Pulsante flottante per tornare in cima
- [x] **Pagefind Search** - Ricerca full-text integrata
- [x] **Sitemap XML** - Per indicizzazione search engine
- [x] **Progress Bar** - Barra di avanzamento lettura articolo
- [x] **Sorting** - Ordinamento articoli per data (recenti/vecchi) e alfabetico
- [x] **Light Mode Search** - Pagefind con colori corretti in light mode

## In Programma

### Content & SEO

- [ ] **RSS Feed** - Feed per lettori RSS/Atom
- [ ] **Open Graph / Twitter Cards** - Preview social quando condividi link
- [ ] **Structured Data (JSON-LD)** - Schema.org per rich snippets Google
- [ ] **Canonical URLs** - Evitare contenuti duplicati

### UI/UX

- [ ] **Breadcrumbs** - Navigazione gerarchica (Home > Articoli > Titolo)
- [ ] **Related Articles** - Suggerimenti articoli correlati in fondo
- [ ] **Tags/Categories** - Sistema di tagging per filtrare contenuti
- [ ] **Keyboard Navigation** - Shortcut (j/k per navigare, / per search)
- [ ] **Print Styles** - CSS ottimizzato per stampa articoli

### Performance

- [ ] **Image Optimization** - Astro Image con lazy loading
- [ ] **View Transitions** - Animazioni fluide tra pagine (Astro 3.0+)
- [ ] **Prefetch Links** - Precaricamento pagine on hover

### Engagement

- [ ] **Newsletter Signup** - Form iscrizione (Buttondown/Substack embed)
- [ ] **Comments** - Sistema commenti (Giscus/GitHub Discussions)
- [ ] **Share Buttons** - Condivisione social (Twitter, LinkedIn, Copy Link)
- [ ] **Reactions** - Like/bookmark articoli (localStorage o GitHub API)

### Developer Experience

- [ ] **Component Showcase** - Pagina demo di tutti i componenti MDX
- [ ] **Storybook-like Preview** - Anteprima componenti isolati
- [ ] **Content Linting** - Validazione frontmatter e link rotti

### Internazionalizzazione

- [ ] **i18n Support** - Versione inglese degli articoli
- [ ] **Language Switcher** - Toggle IT/EN

### Analytics & Monitoring

- [ ] **Privacy-friendly Analytics** - Plausible/Umami/Fathom
- [ ] **Error Tracking** - Sentry per errori runtime
